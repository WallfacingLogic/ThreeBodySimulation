# ThreeBodySimulation

项目名称
--

`ThreeBodySimulation` 是一个用 C++ 编写的三体问题数值模拟程序。该项目目的是演示和研究在经典牛顿引力下的多体轨道演化，并提供可配置的数值积分参数、输出格式以及简单的可视化/导出支持。

主要特性
--

- 基于经典牛顿万有引力的三体动力学模拟
- 可配置的积分器参数（时间步长、总模拟时长、输出频率）
- 支持导出轨道数据为 CSV 以便后续可视化
- 以 `Visual Studio` 项目形式组织，便于在 Windows 上构建和调试

先决条件
--

- Windows 10/11
- `Microsoft Visual Studio`（含 C++ 开发工具）。项目文件：`ThreeBodySimulation.vcxproj`
- （可选）用于后处理和可视化的 Python、Matplotlib 或其它绘图工具

构建与运行
--

1. 使用 Visual Studio 打开解决方案或项目：

   - 双击 `ThreeBodySimulation.vcxproj` 或在 Visual Studio 中选择 "Open -> Project/Solution" 并打开项目文件。

2. 选择合适的配置（Debug/Release）和平台（x86/x64），然后构建项目。

3. 在生成后，运行可执行文件（在 IDE 中按 F5 或从输出目录直接运行）。

命令行/配置
--

项目可以根据实现接收一些命令行参数或在源代码中配置初始条件。常见的配置项：

- 初始位置和速度（每个天体）
- 质量
- 时间步长（`dt`）
- 总模拟时间
- 输出间隔（保存轨道数据的频率）

（说明：具体的参数名称和用法请参阅源码中的 `main` 函数或帮助输出。）

输出格式
--

- 程序将轨道数据导出为 CSV 文件（时间、每个天体的位置坐标），便于用第三方工具制作动画或进行数据分析。
